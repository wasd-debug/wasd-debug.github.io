Vscode快捷键大全

前言

最近我在vscode上配置好了语言环境，决定转战到vscode。所以整理了vscode的快捷键，同时也分享给大家，共同学习吧。其中没有进行制作Gif的快捷键，一部分是因为，我的小键盘上没有，无法录制；有一部分是，我也不知道这个快捷键有什么鸟用……；后面是因为，确实累了。。。能力有限，本文快捷键是由Vscode官方文档翻译而来，有一些翻译不到位和不好理解的地方，请多多包涵。

General

Ctrl+Shift+P 显示命令面板

Ctrl+P 快速打开，进入文件

Ctrl+Shift+N 新窗口

Ctrl+Shift+W 关闭窗口

Ctrl+,用户设置

Ctrl+K Ctrl+S 键盘快捷键

Basic Editing

Ctrl+X 剪切

Ctrl+C 复制行

Alt+↑/↓ 行上下移动

Shift+Alt+↑/↓ 复制行 向上/向下

Ctrl+Shift+K 删除一行

Ctrl+Enter 下面插入一行 Ctrl+Shift+Enter 上面插入行

Ctrl+Shift+\ 跳转到匹配括号

Ctrl + ] / [ 行缩进/减少缩进

Ctrl+↓/↑ 逐行向上/下滚动

Ctrl+Shift+[ 折叠区域 Ctrl+Shift+] 展开区域

Ctrl+K Ctrl+0 折叠所有区域 Ctrl+K Ctrl+J 展开所有区域

Ctrl+K Ctrl+C 注释该行 Ctrl+K Ctrl+U 不再注释该行

Ctrl+/ 切换该行的注释（注释与否） Shift+Alt+A 切换块注释（选中区域后，注释该区域或取消注释）

Alt+Z 切换自动换行

Ctrl+K Ctrl+[ 折叠所有子区域

Ctrl+K Ctrl+] 展开所有子区域

Alt+PgUp/PgDn 向上/向下滚动页面

Home / End 转到行首/行尾

Ctrl + Home 转到文件开头

Ctrl+ End 转到文件的末尾

Navigation

Ctrl+T 显示所有符号

Ctrl+G 跳转到行

Ctrl +P 跳转到文件

Ctrl+Shift +O 跳转到某函数

Shift + F8 转到以前的错误或警告

Ctrl+Shift+Tab 切换编辑器

Ctrl+Shift+M 显示问题面板

Alt + ← / → 后退 / 前进

Ctrl+M 移动焦点

F8 转到下一个错误或警告

查找替换

Ctrl+F 查找

Ctrl+H 替换

F3/Shift+F3 查找上/下一个

Alt+Enter 选中所有查找匹配中出现的 Ctrl+D 将光标移动到下一个查找匹配

Alt+C/R/W 切换区分大小写/正则表达式/整个单词

多光标和选择

**Alt+左键单击 增加光标 Ctrl+Alt+↑/↓ 在上方和下方增加光标 Shift+Alt+l 在选中的每一行末尾加入光标 **

Shift+Alt+(拖动鼠标) 在一列中增加光标 Ctrl+Shift+Alt+(箭头键) 在一列中增加光标

Ctrl+U 撤销最后一个光标操作

Ctrl+L 选中当前行

Shift+Alt+→ 扩大选择 Shift+Alt+← 缩小选择

Ctrl+Shift+L 选中所有匹配项

Ctrl+F2 选中当前所有匹配单词

丰富的语言编辑

Ctrl+Shift+空格 触发参数提示

F12 转到定义 Alt+F12 速览定义 Ctrl+K F12 在一侧打开定义

Shift+F12 展示引用

F2 重命名符号

Ctrl+K Ctrl+X 修剪空白

Ctrl+K M 更改文件语言

Ctrl+. 快速修复

Shift+alt+F 格式文档

Ctrl+K Ctrl+F 格式选择

Ctrl+空格 触发建议

编辑器管理

Ctrl+K F 关闭文件夹

Ctrl+\ 再打开一个编辑器

Ctrl+1/2/3 关注第一个第二个或第三个编译器

Ctrl+K Ctrl+←/→ 关注前一个/后一个编辑器

Ctrl+K ←→ 移动活跃编辑器

Ctrl+W 关闭编辑器

Ctrl+shift+PgUp/PgDn 编辑器左右移动

文件管理

Ctrl+N 新建文件

Ctrl+O 打开文件

Ctrl+shift+S 另存为

Ctrl+Shift+Tab 打开前一个标签

Ctrl+K P 复制当前文件夹的路径

Ctrl+K R 在资源管理器中显示当前文件

Ctrl+K O 在新窗口显示当前文件

Ctrl+F4 关闭

Ctrl+K Ctrl+W 关闭所有

Ctrl+Shift+T 重新打开关闭的文件

Ctrl+K 回车 保持预览模式编辑器打开

Ctrl+Tab 打开下一个

Ctrl+K S 全部保存

Ctrl+S 保存

展示

F11 切换全屏

Shift+Alt+0 切换编辑器布局（水平、竖直）

Ctrl+ =/- 放大缩小

Ctrl+B 切换栏的可见性

Ctrl+Shift+H 在文件中替换

Ctrl+Shift+U 展示输出面板

Ctrl+Shift+V 打开Markdown预览

Ctrl+K V打开Markdown预览在一侧

Ctrl+K Z 禅模式

Ctrl+shift+E 显示浏览器\切换焦点

Ctrl+Shift+G 显示源代码管理栏

Ctrl+shift+D 显示调试栏

Ctrl+shift+X 显示扩展栏

Ctrl+Shift+J 切换搜索细节

调试

F9 切换断点

F5 开始\继续

Shift+F5 停止

F11/Shift+F11 跳入跳出

F10 跳过

Ctrl+K Ctrl+I 显示循环

集成终端

Ctrl+↑/↓ 向上滚动

Shift+PgUp/PgDn 向上滚动一页（下）

Ctrl+Home/End 滚到顶部、底部

Ctrl+C 复制

Ctrl+V 粘贴

Ctrl+` 显示综合终端

Ctrl+SHift+` 创建新的终端
